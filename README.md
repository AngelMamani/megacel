# admin-frontend

Panel de administración para MEGA CEL. React + TypeScript + Vite + Firebase (Auth, Firestore, Storage).

## Requisitos

- Node.js 18+
- Cuenta Firebase con proyecto configurado

## Uso

```bash
npm install
cp .env.local.example .env.local   # Configurar VITE_FIREBASE_*
npm run dev
```

Build: `npm run build`
